# PHP_Learning_Journal

**PHP_Learning_Journal** 

This learning journal allows the user record their day to day learning progress. The learning joural performs CRUD. Key function includes create entry and tag, update entry and tag, delete entry and tag, and of course read them! Hope you like it and happy coding!
